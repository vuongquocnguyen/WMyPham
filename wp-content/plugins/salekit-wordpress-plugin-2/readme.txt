== Installation ==



It can also be installed manually.

1. Download the Web Push Notification plugin from WordPress.org.

2. Unzip the package and move to your plugins directory.

3. Log into WordPress and navigate to the Plugins screen.

4. Locate Web Push Notification in the list and click the *Activate* link.

5. You will be redirected to the Web Push Notification Getting Started page.
